# Play Store listing draft

## App name

- **SK School Master**

## Short description (max ~80 chars)

All-in-one school management for attendance, fees, homework, exams & reports.

## Full description

SK School Master helps schools manage day-to-day operations with speed and clarity.

Admins can:

- Manage teachers and students
- Create announcements
- Track attendance
- Manage fees
- Organize exams (Exam Types → Exams → Marks)
- View reports (attendance, fees, exams, student report)

Teachers can:

- View assigned classes
- Mark attendance for their classes
- Add homework
- View announcements

Parents can:

- Access their child’s data (when parent linking is enabled by the school)
