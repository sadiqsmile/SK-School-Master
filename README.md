# SK School Master

SK School Master is a school management app built with Flutter + Firebase.

Key modules include attendance, homework, fees, exams/results, announcements, and reports with role-based access (super admin, school admin, teacher, parent).

## Getting Started

### Prerequisites

- Flutter SDK (stable)
- A Firebase project configured for Android/iOS/Web (see `lib/firebase_options.dart`)

### Run locally

1) Install dependencies

`flutter pub get`

2) Run the app

`flutter run`
